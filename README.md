# Personal Website Draft

This is a lightweight single-page website for Prachi Sharma.

## Files

- `index.html`: page structure
- `styles.css`: styling and responsive layout
- `script.js`: all editable content in one place
- `assets/portrait-placeholder.svg`: replace with your main portrait
- `assets/life-placeholder.svg`: replace with a personal/lifestyle image

## How To Personalize

1. Replace the placeholder images:
   - Put your main headshot at `assets/portrait-placeholder.svg` or update the image path in `index.html`
   - Put a personal image at `assets/life-placeholder.svg` or update the image path in `index.html`
2. Open `script.js` and update:
   - Hero title and summary
   - About paragraphs
   - Services
   - Projects
   - Writing entries and links
   - Contact email and location
3. If you want to add more projects or blogs, duplicate an item in the relevant array inside `script.js`

## Best Content To Send Next

- Your preferred email address
- 1 professional headshot
- 1 optional candid/personal image
- 3 to 5 real projects or achievements to feature
- 2 to 3 blog titles with links or drafts
- A short personal note you are comfortable sharing
- Any specific colors or style preferences
